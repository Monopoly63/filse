ما تم تغييره في ml_agent.py

بالعربي:
commands ح لو细胞内المستخدم. كل مرة تشغل السكربت، بيشتغل تدريب كامل من الصفر.
In English:
The file was upgraded for repeated training. Each run is a full new training cycle. Models train, tune, and compare themselves every time.

كيف تشغل التدريب؟ / How to run

افتح PowerShell أو CMD في مجلد المشروع واكتب:
python ml_agent.py
Open PowerShell/CMD in the project folder and type:
python ml_agent.py

الميزات الجديدة / New Features

  • نماذج أقوى: RandomForest + GradientBoosting + ExtraTrees
  • بحث عن أفضل إعدادات: RandomizedSearchCV يبحث عن أفضل قواعد تلقائي
  • تجربة متعددة: 5-fold cross-validation لكل نموذج
  • دمج النماذج: VotingEnsemble + StackingRegressor
  • حفظ التاريخ: training_history.json يحفظ كل run
  • تقرير HTML: ml_report.html يتحدث تلقائي
  • Stronger models: RandomForest + GradientBoosting + ExtraTrees
  • Auto-tuning: RandomizedSearchCV finds best settings
  • Cross-validation: 5-fold CV for each model
  • Ensembles: VotingEnsemble + StackingRegressor
  • History saved: training_history.json keeps all runs
  • HTML report: ml_report.html updates automatically

الملفات الناتجة / Output Files

كل تشغيل ينتج ملفات في مجلد outputs/:
Every run creates files in outputs/:
  • ml_report.html — التقرير الكامل / Full report
  • model.pkl — أفضل نموذج / Best model
  • training_history.json — سجل كل Runs / History of all runs
  • predicted_actual.png — رسم بياني / Chart
  • feature_importance.png — أهمية الخصائص / Feature importance chart

ملاحظة / Note

السكربت يحتاج إنترنت first time لتحميل بيانات من Kaggle. إذا ما كان إنترنت، بيستخدم بيانات صناعية ( synthetic ) والتدريب يشتغل عادي.
The script needs internet first time to download from Kaggle. If no internet, it uses synthetic data and training still works.
